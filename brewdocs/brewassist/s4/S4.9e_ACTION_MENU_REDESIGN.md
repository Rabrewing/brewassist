## S4.9e — ActionMenu Redesign

### A. Role of the + Menu (what it *should* be)

**Purpose:**
Per-message helpers only — *not* a second MCP sidebar.

**Allowed items:**

* Upload image / screenshot
* Take photo (camera)
* Import from Drive
* “Use HRM Deep Reasoning for this message”
* “Send this message to NIMs Research Tier”

**Not allowed:**

* File operations
* Git Command Center
* Database Assistant
* Anything that’s already a big MCP tool on the left

If we ever want shortcuts, it should be **one** row:

> “Open MCP Tools Panel”

…not a full duplicate.

---

### B. Visual Redesign

**Container:**

* Anchor: below the **+** button, left-aligned with the chat box
* Max width: `320px`
* Background: `var(--brew-card)` (deep gray / #232323)
* Border: `1px solid var(--brew-border-soft)` (subtle teal)
* Shadow: soft glow, like `0 0 18px rgba(0, 199, 183, 0.18)`
* Border radius: `24px`
* Padding: `10px 8px`

**Item layout:**

Each action row:

* Height ~ `44px`
* Display: `flex; align-items: center; justify-content: space-between;`
* Left:

  * Label (LED white, `font-weight: 600`, `font-size: 0.86rem`)
  * Subtext (dim grey, `font-size: 0.72rem`)
* Right:

  * Tiny icon (camera, file, brain, etc.) in a **BrewTeal** circle

**States:**

* Default:

  * Background: transparent
  * Label: LED white
* Hover:

  * Background: `rgba(0, 199, 183, 0.12)`
  * Left border accent: `2px solid #00C7B7`
* Active/click:

  * Brief invert glow around the row

---
