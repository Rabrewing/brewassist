### B. Visual Redesign

**Container:**

- Anchor: below the **+** button, left-aligned with the chat box
- Max width: `320px`
- Background: `var(--brew-card)` (deep gray / #232323)
- Border: `1px solid var(--brew-border-soft)` (subtle teal)
- Shadow: soft glow, like `0 0 18px rgba(0, 199, 183, 0.18)`
- Border radius: `24px`
- Padding: `10px 8px`

**Item layout:**

Each action row:

- Height ~ `44px`
- Display: `flex; align-items: center; justify-content: space-between;`
- Left:
  - Label (LED white, `font-weight: 600`, `font-size: 0.86rem`)
  - Subtext (dim grey, `font-size: 0.72rem`)

- Right:
  - Tiny icon (camera, file, brain, etc.) in a **BrewTeal** circle

**States:**

- Default:
  - Background: transparent
  - Label: LED white

- Hover:
  - Background: `rgba(0, 199, 183, 0.12)`
  - Left border accent: `2px solid #00C7B7`

- Active/click:

- Brief invert glow around the row

### D. Slash Command Palette

The center composer should also expose a small `/` command palette for power users.

- Trigger: typing `/` in the composer, or clicking the `Commands` button.
- Purpose: quick insert/run of high-frequency commands like `/init`, `/task`, `/doc`, `/patch`, `/hrm`, `/llm`, `/agent`, and `/loop`.
- This is separate from the `+` helper menu and should stay compact.
