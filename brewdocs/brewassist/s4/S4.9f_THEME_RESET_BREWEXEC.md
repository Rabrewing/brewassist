## S4.9f — BrewTheme Reset

We kill the navy and go back to your **BrewExec palette**.

### A. Core tokens

Define these CSS variables (either in `:root` or `.brew-shell`):

```css
:root {
  --brew-bg: #181818;              /* main background */
  --brew-bg-elevated: #232323;     /* cards, bubbles */
  --brew-bg-panel: #1c1c1c;        /* side panels */
  --brew-border-soft: rgba(0, 199, 183, 0.5);

  --brew-gold: #ffd700;
  --brew-teal: #00c7b7;
  --brew-white: #fdfdfd;
  --brew-muted: #aaaaaa;

  --brew-tab-bg: #101010;
  --brew-tab-active-bg: #232323;
  --brew-tab-active-border: #ffd700;

  --brew-shadow-gold: 0 0 18px rgba(255, 215, 0, 0.18);
  --brew-shadow-teal: 0 0 22px rgba(0, 199, 183, 0.22);
}
```

### B. Layout surfaces

In `cockpit-base.css` / `cockpit-layout.css`:

```css
body,
.brew-shell {
  background: radial-gradient(circle at top left, #202020 0%, #000 60%);
  color: var(--brew-white);
}

.brew-header {
  background: linear-gradient(90deg, #181818, #232323);
  border-bottom: 1px solid rgba(255, 215, 0, 0.2);
}

.brew-main {
  background: transparent; /* let cosmic bg show */
}

.brew-sidebar-left,
.brew-sidebar-right {
  background: var(--brew-bg-panel);
  box-shadow: var(--brew-shadow-teal);
}
```

### C. Mode tabs (HRM / LLM / AGENT / LOOP)

Old-style “pills”:

```css
.brew-mode-pill {
  background: var(--brew-tab-bg);
  border-radius: 999px;
  padding: 6px 14px;
  font-size: 0.8rem;
  letter-spacing: 0.04em;
  color: var(--brew-muted);
  border: 1px solid rgba(255, 255, 255, 0.04);
}

.brew-mode-pill.active {
  background: var(--brew-tab-active-bg);
  border-color: var(--brew-tab-active-border);
  color: var(--brew-gold);
  box-shadow: var(--brew-shadow-gold);
}
```

You can tweak each mode with a subtle glow:

```css
.brew-mode-pill.hrm.active { border-color: #ffd700; }
.brew-mode-pill.llm.active { border-color: #00c7b7; }
.brew-mode-pill.agent.active { border-color: #ff5a5f; }
.brew-mode-pill.loop.active { border-color: #9c27b0; }
```
