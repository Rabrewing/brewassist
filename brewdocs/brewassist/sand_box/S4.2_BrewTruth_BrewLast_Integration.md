## 1. Purpose

S4.2 integrates the BrewTruth Engine with the BrewLast logging system. The primary goal is to enable BrewTruth to *review* toolbelt actions recorded in BrewLast and attach a `truthReview` block onto that run in `.brewlast.json`. This establishes a feedback loop where BrewAssist actions are not only logged but also evaluated for their safety and correctness.

---

## 2. Final Design

### 2.1 Extend BrewLast Schema

The `BrewLastToolRun` type in `lib/brewLast.ts` was extended to include an optional `truthReview` field. A new type, `BrewTruthReview`, was defined to encapsulate the evaluation results from the BrewTruth Engine.

```typescript
export type BrewTruthReview = {
  truthScore: number;        // 0–1
  riskLevel: "LOW" | "MEDIUM" | "HIGH";
  flags: string[];           // any issues / warnings
  summary: string;           // short human summary
};

export type BrewLastToolRun = {
  // ... existing fields ...
  truthReview?: BrewTruthReview; // 🔥 NEW: optional truth review attached later
};
```

### 2.3 New API Endpoint: `/api/brewtruth-from-last`

A new API endpoint, `pages/api/brewtruth-from-last.ts`, was created. This endpoint serves as the orchestrator for the BrewTruth review process on BrewLast entries.

**Functionality:**
1.  Reads the current `.brewlast.json` state.
2.  Identifies the `lastToolRun` (or a specific `toolRunId` if provided in the request).
3.  Calls `runTruthCheckForToolRun` (from `lib/brewtruth.ts`) to obtain a `truthReview` for the selected tool run.
4.  Updates the BrewLast state by attaching the `truthReview` to both `lastToolRun` and the corresponding entry in the `history` array.
5.  Saves the updated BrewLast state back to `.brewlast.json`.
6.  Returns the `toolRunId` and the generated `truthReview`.

**Key Fixes during implementation:**
*   Corrected import and usage of `readBrewLast` and `writeBrewLast` in `pages/api/brewtruth-from-last.ts` (originally `loadBrewLast` and `saveBrewLast` were specified, but `lib/brewLastServer.ts` exports `readBrewLast` and `writeBrewLast`).
