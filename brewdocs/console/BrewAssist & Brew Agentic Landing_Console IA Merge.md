3\. Design System Rules For The Console

Backgrounds

near-black base with subtle gradient depth

avoid busy cosmic backgrounds inside the app shell

reserve special glow only for hero panels or live status moments

Panels

dark glass cards

20–24px radius on major surfaces

thin gold or slate borders depending on importance

Accent Logic

gold \= premium, navigation highlights, trust, billing, ownership

teal/cyan \= live, connected, active, AI, routing, telemetry

red \= blockers / risk

muted slate \= inactive / unavailable

Typography

strong headline weight for dashboard titles

clean UI body font for density

compact uppercase eyebrow labels for section grouping

Motion

smooth fades and subtle slide-ups

no hyperactive animation

use motion to communicate state changes and hierarchy
