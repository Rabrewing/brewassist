## 2️⃣ Add Truth Badge in the Header Area
In `BrewCockpitCenter.tsx`, where you currently render header/status:

* Extend the message state to include `truth` from `/api/brewassist`.
* Compute a tier badge:

```ts
function getTruthBadge(truth?: BrewTruthReport | null) {
  if (!truth) return { label: "Ungraded", className: "truth-pill-neutral" };

  switch (truth.tier) {
    case "gold":
      return { label: "Truth: Gold", className: "truth-pill-gold" };
    case "silver":
      return { label: "Truth: Silver", className: "truth-pill-silver" };
    case "bronze":
      return { label: "Truth: Bronze", className: "truth-pill-bronze" };
    case "red":
      return { label: "Truth: Red Flag", className: "truth-pill-red" };
    default:
      return { label: "Truth: Unknown", className: "truth-pill-neutral" };
  }
}
```

In JSX:

```tsx
const badge = getTruthBadge(lastMessage.truth);

<div className="cockpit-status-row">
  <div className={`truth-pill ${badge.className}`}>{badge.label}</div>
  <div className="truth-meta">
    {lastMessage.truth?.riskSummary || "No additional risk summary."}
  </div>
</div>
```

## 3️⃣ Auto-Scroll Fix in Message Log

In `BrewCockpitCenter.tsx`:

```tsx
const logRef = useRef<HTMLDivElement | null>(null);

useEffect(() => {
  if (!logRef.current) return;
  logRef.current.scrollTop = logRef.current.scrollHeight;
}, [messages]);
```

And in JSX:

```tsx
<div ref={logRef} className="cockpit-message-log">
  {messages.map(...)}
</div>
```

Ensure CSS:

```css
.cockpit-message-log {
  flex: 1 1 auto;
  overflow-y: auto;
  padding: 0.75rem 1rem 1rem;
}
```
